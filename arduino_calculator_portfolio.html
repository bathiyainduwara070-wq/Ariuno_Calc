Arduino Calculator

by Bhathiya Induwara Jayasena

This project is an Arduino-based calculator using a 4x4 keypad and I2C LCD display. Users can perform addition, subtraction, multiplication, and division operations with real-time feedback.

Arduino UNO I2C LCD 16x2 4x4 Keypad
01 — OVERVIEW

What does this project do?

This project is a physical Arduino-based calculator connected with a 4x4 keypad and 16x2 I2C LCD display. Users can enter numbers and perform mathematical calculations directly from the keypad while results are displayed instantly on the LCD screen.

🧮

Real-Time Operations

The calculator performs basic arithmetic operations including addition, subtraction, multiplication, and division. Users enter values through the keypad and instantly receive results.

⌨️

4x4 Keypad Input System

A membrane keypad is used as the main input device. Numeric buttons are used for entering numbers while special buttons control operations (A: +, B: -, C: ×, D: /, #: Calculate, *: Clear).

📟

I2C LCD Display Output

The 16x2 I2C LCD provides real-time output for user input, operations, and final calculation results. The display also shows a custom startup screen with the project name.

⚡

Automatic Reset Function

After displaying the final answer, the calculator automatically clears previous values and returns to the start screen, allowing users to begin a new calculation quickly.

System Operation Steps

  1. User enters numbers using the keypad.
  2. Arithmetic operations are selected using A, B, C, or D buttons.
  3. The LCD displays entered values and operations in real time.
  4. Pressing # calculates the final result.
  5. The Arduino processes the calculation and displays the answer on the LCD.
  6. After a short delay, the calculator automatically resets for the next calculation.
04 — SOFTWARE

Arduino & Embedded System

This project uses an Arduino sketch developed in the Arduino IDE to control the keypad input system, perform mathematical calculations, and display results on the I2C LCD screen.

🔧

Arduino Sketch Features

  • • Initializes the I2C 16x2 LCD display
  • • Reads user input from the 4x4 membrane keypad
  • • Supports addition, subtraction, multiplication, and division
  • • Displays live keypad input on the LCD
  • • Calculates results when the # button is pressed
  • • Automatically clears and resets after calculation
  • • Sends results through Serial at 9600 baud rate
  • • Custom startup screen: "Bhathiya's Calculator"
⌨️

Keypad Control System

Numeric buttons capture input. Special logic routes A, B, C, D to operation handlers. # executes the math, and * clears the active buffer to reset.

📟

I2C LCD Display System

Displays startup welcome screen, live buffer of entered numbers and operators, and the final results. Uses I2C (SDA/SCL) for simplified 2-wire data communication.

07 — REFLECTION

What I Learned

A short reflection on the development process, challenges, and results.

One of the most important parts of this project was learning how to build and program a complete Arduino-based embedded system using real hardware components. Connecting the I2C LCD display and 4x4 keypad correctly required careful wiring and understanding of Arduino pin communication.

The most challenging part of the project was configuring the keypad operations and managing the calculator logic properly. Understanding how to detect keypad input, store multiple numbers, process mathematical operations, and reset the system after calculations was an important learning experience. Working with the I2C LCD display helped me understand how serial communication protocols simplify hardware connections by reducing the number of wires needed between components and the Arduino board.

This project also improved my programming skills in Arduino C++. I learned how to:

  • Read keypad input and control LCD output
  • Create functions for calculations
  • Manage variables and user input
  • Use conditional statements for operation handling
  • Implement automatic system reset behavior

The final version of the project works successfully:

  • The keypad responds accurately to user input
  • Mathematical calculations execute correctly
  • Results display clearly on the LCD screen
  • The calculator automatically resets after calculations
  • Serial communication outputs results through the USB connection

Overall, this project provided practical experience with embedded systems, hardware interfacing, and Arduino software development while demonstrating a complete working calculator system using physical electronic components.
